# DynamicKeycards-LabAPI

Plugin for creating custom keycards.

View the example config [here](https://github.com/TeamEXAngus/DynamicKeycards-LabAPI/blob/master/example-config.yml).

Click [here](https://github.com/TeamEXAngus/DynamicKeycards-For-Dynamic914-LabAPI) for integration with [Dynamic 914](https://github.com/TeamEXAngus/Dynamic914-LabAPI).

## Console Commands
`givekeycard <player id> <keycard name>`
e.g. `givekeycard 12 credit_card`
